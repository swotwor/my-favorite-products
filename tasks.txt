Список покупок
1. По клику на карточку продукта, он должен добавляться в список покупок.
2. Когда вводим количество продукта для покупок, он так же должен добавляться в список покупок.
3. Продукт должен удаляться из списка опкупок по клику на его карточку.
4. В режиме редактирования списка покупок, должна быть возможность редактирования количество покупаемого продукта.
5. При редактировании списка покупок, должна быть возможность добавить и другие продукты. 

Категории товаров
1. Категории будут в виде спойлеров, по нажатию на которые будет выпадать список продуктов.
2. Напротив названия каждой категории будет отображаться количество продуктов в данной категории.
3. При создании списка, продукты будут отображаться в категориях и рядом с количеством продуктов в скобках будет отображаться и количество добавленных продктов в список из этой категории.
